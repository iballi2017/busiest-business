How To Implement Page-loader In Your Project Using Javascript and jQuery

Step 1: Get a Spinner or animation component for your project

In this project, I used a group of Bootstrap 4 Spinners. You can use one depending on your preference.
Create a div tag to contain your spinner and give it a className. In this project, I named it "preload". Place this div container and its content immediately after your opening body tag. Then give it style like below:
.preload { position: fixed; top: 0; width: 100%; height: 100vh; background: #202020; z-index: 2000; display: flex; justify-content: center; align-items: center; transition: opacity 0.5s ease; } .preload-finish { opacity: 0; pointer-events: none; }
NOTE: "preload-finish", is a class that makes "preload" disappears when it is attached. Javascript will do the attachment dynamically.
Add the following JavaScript codes;
option 1:
$(window).on("load", function () { $(".preload").hide("slow") })

This code doesn't need you to create a "preload-finish" class

option 2:
$(".preload").addClass("preload-finish")

This code will append "preload-finish" class when the page loads

option 3:
window.addEventListener("load", () => { const preload = document.querySelector(".preload") preload.classList.add("preload-finish") })

This code will also append "preload-finish" class when the page loads

You can also use this method of preloader at different sections of your page. For example, if you're fetching images/videos from a CDN, it might take time for the image to arrive. You only need to remove "window" from your code and replace with the class or id of the container element. $(".image-container").on("load", function () { $(".preload").hide("slow") })

How To Test

You can see the result of a preloader when you reload your page. If your network is very fast, you might not notice it. You will definitely notice if you have a poor network.

Hard-refresh your page with "Ctrl+F5" or "Ctrl+Shift+R", and you'll see your preloaded animating, waiting for your page to load before it disappears.
